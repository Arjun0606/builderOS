## üõ†Ô∏è **HOW TO ADD A NEW TEMPLATE**

### **Step 1: Define Template**
Add to `COMPREHENSIVE_TEMPLATES` array in `app/lib/templates-full.ts`:

```typescript
{
  id: 'unique-template-id',
  name: 'Human-Readable Template Name',
  category: 'One of TEMPLATE_CATEGORIES',
  practiceArea: 'One of PRACTICE_AREAS',
  description: 'Brief description of what this template does',
  variables: ['VAR1', 'VAR2', 'VAR3'], // Placeholders in [BRACKETS]
  tags: ['keyword1', 'keyword2', 'keyword3'], // For search
  content: `
FULL TEMPLATE CONTENT HERE

Use [VARIABLE_NAME] for placeholders.

Include proper legal formatting.
Include relevant section references.
Include verification clauses where needed.
  `,
}
```

### **Step 2: Test Template**
1. Go to `/dashboard/templates`
2. Search for your new template
3. Click "Generate"
4. Fill in variables
5. Review output

### **Step 3: Get Legal Review**
Have a lawyer review the template for:
- Legal accuracy
- Proper terminology
- Section references
- Formatting

## üí° **TEMPLATE CONTENT GUIDELINES**

### **1. Legal Accuracy**
- Use correct section references
- Use proper legal terminology
- Follow latest amendments

### **2. Formatting**
- Use CAPS for headers
- Use proper indentation
- Use numbering for clauses
- Include verification/declaration sections

### **3. Variables**
- Use [BRACKETS] for variables
- Use DESCRIPTIVE names (not VAR1, VAR2)
- Group related variables
- Provide clear variable descriptions

### **4. Flexibility**
- Make templates adaptable to different scenarios
- Include optional clauses
- Allow AI refinement with custom instructions
